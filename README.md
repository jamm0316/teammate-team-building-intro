# 팀메이트 팀빌딩 소개

### 👉 [지금 바로 팀메이트 둘러보기](https://jamm0316.github.io/teammate-team-building-intro/)

> 위 링크를 누르면 팀메이트 팀빌딩 소개 페이지로 이동합니다.

---

## 소개

**팀메이트(Teammate)** 는 주니어 백엔드 개발자가 혼자가 아니라 **함께** 성장하도록 돕는 팀빌딩·스터디 프로그램입니다.
이 저장소는 그 여정을 담은 소개 페이지를 [GitHub Pages](https://jamm0316.github.io/teammate-team-building-intro/)로 배포한 결과물입니다.

## 무엇을 함께 하나요

- **📚 커리큘럼** — 방향 잡기부터 시작해 단계별로 따라가는 5개월의 성장 로드맵
- **📝 이력서·포트폴리오 다듬기** — 서로의 이력서를 보며 공통점을 찾고, 버전 관리를 통해 변화 과정을 기록
- **📖 함께 읽는 책** — *주니어 백엔드 개발자가 반드시 알아야 할 실무 지식* 등 함께 읽고 감을 잡는 도서 (책 목록은 변경될 수 있습니다)
- **🌍 팀메이트 월드** — 캐릭터 뽑기, 실시간 라이브, 쪽지 사물함 등을 담은 팀 프로젝트 결과물

## 페이지 살펴보기

소개 페이지는 다음과 같은 흐름으로 구성되어 있습니다.

1. 방향 잡기 — 무엇을, 왜 하는지 함께 그려보기
2. 커리큘럼 — 주차별로 진행되는 활동
3. 이력서·포트폴리오 리뷰 — 실제 검토 현장 사진과 함께
4. 팀메이트 월드 — 팀 프로젝트 갤러리

각 섹션의 사진 갤러리는 마우스 드래그로 가로 스크롤하며 둘러볼 수 있습니다.

## 직접 확인하기

이 저장소를 클론한 뒤 `index.html`을 브라우저에서 열면 동일한 페이지를 로컬에서 확인할 수 있습니다.

```bash
git clone https://github.com/jamm0316/teammate-team-building-intro.git
cd teammate-team-building-intro
open index.html   # macOS · Windows는 start, Linux는 xdg-open
```
